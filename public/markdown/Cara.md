## ðŸš€ Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/chakravarthigit/cara-frontend.git
   cd cara-frontend
   ```

2. **Install dependencies:**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Run the app (Android/iOS):**
   ```bash

   ```

   ```bash
   npm test

   yarn test
   ```
